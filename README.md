# nuxtAccordingToUseElement

> nuxt 按需引入 element 和 cookie验证
> 在打包的时候最好使用npm run generate 而不是 npm run build
> 因为在使用runbuild的时候打包出来的vendor会被带入到浏览器缓存中
> 会显示为原来的样式

## Build Setup

``` bash
$ yarn or npm install

$ yarn or npm run dev

$ yarn or npm run build
$ yarn or npm start

$ yarn or npm run generate
```
